# 🚀 About Ishify

Ishify is a modern and interactive music streaming web application created using **HTML, CSS and JavaScript**.

The project focuses on creating a stylish frontend music experience with dynamic JavaScript functionalities, responsive UI and interactive music controls.

---

# ✨ Features

## 🎵 Music Player
- Play / Pause Songs
- Next / Previous Controls
- Song Progress Bar
- Volume Slider
- Playback Speed Control
